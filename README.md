# Secure Legacy Data Service

A server-side Java implementation demonstrating robust data access patterns and session management. 

## Technical Highlights
* **DAO Pattern:** Implemented the **Data Access Object (DAO)** pattern to decouple business logic from the underlying HSQLDB database logic.
* **Session Management:** Utilized Java Servlets to handle secure user sessions and state management across the web lifecycle.
* **CRUD Logic:** Features a complete implementation of Create, Read, Update, and Delete operations with input sanitization.

## Tech Stack
* **Language:** Java
* **Server:** Tomcat / Java Servlets
* **Database:** HSQLDB
